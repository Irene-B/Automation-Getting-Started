# Example Project with Winium
a simple calculator with allure report

# Dependency 
Java 
Junit
winium
winium Desktop APP 

# Testing command 

- First you need to run Winium Server , there is a command file "run_driver.cmd". 
  RUN THIS COMMAND to run your server. 
  
  Then use maven to test 

- mvn clean test
